\doxysubsection{Detailed Description}
This class represents a group of multimedia objects. 

\mbox{\hyperlink{classGroupMultimedia}{Group\+Multimedia}} extends the standard list class to hold a collection of multimedia objects. Each multimedia object is managed through shared\+\_\+ptr, facilitating automatic memory management. The class provides functionalities to set and get the group\textquotesingle{}s name and to display information about all objects in the group.
