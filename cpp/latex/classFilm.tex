\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classFilm}{Film}} class extends the \mbox{\hyperlink{classVideo}{Video}} class with specific attributes for films. 

This class represents a film with attributes like the number of chapters and their respective lengths. It provides functionalities to set and get these attributes, display film information, and play the film.
