Detailed Description

This class represents a group of multimedia objects.

GroupMultimedia extends the standard list class to hold a collection of multimedia objects. Each multimedia object is managed through shared_ptr, facilitating automatic memory management. The class provides functionalities to set and get the group's name and to display information about all objects in the group.
