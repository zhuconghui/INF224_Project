Detailed Description

The main window frame of the application, managing the GUI components and client interactions.

MainFrame extends JFrame and is the primary window for the application. It contains the main panel, menu bar, and handles the client-server communication. It is responsible for initializing the GUI components and managing the lifecycle of the application.
