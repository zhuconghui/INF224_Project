\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classMainPanel}{Main\+Panel}} class represents the main panel within the application window. It extends JPanel and contains a \mbox{\hyperlink{classTextAreaPanel}{Text\+Area\+Panel}} and a \mbox{\hyperlink{classButtonPanel}{Button\+Panel}}.
